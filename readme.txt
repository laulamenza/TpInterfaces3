Trabajo entregable Nº 2

Trabajo requerido:
                  Desarrollar una aplicación web similar al clásico Paint de Windows que provea la siguiente
                  funcionalidad:
                                1. Barra de herramientas con, lápiz (que pueda elegir color del lápiz) y goma de borrar,
                                   y su funcionalidad.
                                2. Permitir iniciar con un lienzo en blanco, o partir de una imagen que será
                                   cargada desde disco (Usar un diálogo para elegir la imagen)
                                3. Aplicar 4 filtros a la imagen actual: negativo, brillo, binarización, sepia, etc.
                                4. Permitir guardar en disco la imagen, descartar la imagen y comenzar con un lienzo
                                   vacío.
                                5. La aplicación web debe estar contenida en una página web acorde con la temática de la 
                                   entrega.
                                6. El código debe tener todos sus métodos comentados, dejando así en claro qué
                                   funcionalidad se está implementando y cómo se desarrolla.
Promoción:
           Aplicar 2 de los siguientes filtros a la imagen: Saturación, Detección de Bordes, Blur, otros.
